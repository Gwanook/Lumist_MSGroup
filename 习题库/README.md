# 路觅教育数学与统计学学科组习题库

各位老师可以在这个文件夹下添加课程文件夹，然后再添加学校文件夹（格式：`国家: 学校_课号`）。上传题目时需以`.pdf`、`.docx`、`.md`为拓展名上传，并给这个文件起一个清楚的名字。在上传slides、notes、pastpaper、exercise、problem等，如果文件数>1最好可以新建文件夹插入。对于非学校特有的题目直接放在课程文件夹下即可，文件命名需写清楚文件内容，中英均可。
