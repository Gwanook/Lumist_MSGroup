# 路觅教育数学与统计学学科组习题库

各位老师可以在这个文件夹下添加课程文件夹，然后再添加学校文件夹（格式：`国家: 学校`）。上传题目时需以`.pdf`、`.docx`、`.md`为拓展名上传，并给这个文件起一个清楚的名字，比如“Past paper 2025”。对于非学校特有的题目直接放在课程文件夹下即可，文件命名需写清楚文件内容，中英均可。若想上传lecture notes或slides禁止分章节上传，需整合完整再上传。
